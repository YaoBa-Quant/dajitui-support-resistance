# Contributing

欢迎通过 Issue 和 Pull Request 一起完善这个项目。

## 提交建议

- Bug 反馈请尽量附复现步骤、输入数据特征和报错信息
- 新功能建议请说明使用场景、预期行为和输出形式
- 若修改核心算法，请同步说明假设、边界条件和影响范围
- 若修改研究口径，请明确变更的是数据处理、特征定义还是评估方法

## 推荐流程

1. Fork 本仓库
2. 新建功能分支
3. 提交代码与说明
4. 发起 Pull Request

## 代码与文档约定

- 尽量保持模块职责清晰
- 不提交本地缓存、研究输出和个人数据
- 修改用户可见功能时，尽量同步更新 `README.md`
- 如变更研究结果解释口径，建议同步补充示例或说明

## 测试建议

提交前建议至少执行：

```bash
pytest -q
```

如果修改了数据处理或建模流程，建议附上最小可复现示例。
